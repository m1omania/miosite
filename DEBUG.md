# Отладка проблемы "пустая страница"

## Что проверено:
✅ HTML страница загружается (curl показывает HTML)
✅ Компоненты созданы
✅ Импорты исправлены
✅ API URL обновлен на порт 4001
✅ .env.local создан

## Возможные причины:

1. **JavaScript не выполняется** - проверьте консоль браузера (F12)
2. **Backend не запущен** - убедитесь что backend работает на порту 4001
3. **Проблемы с Tailwind CSS** - проверьте что стили применяются

## Проверка:

1. Откройте консоль браузера (F12 → Console)
2. Проверьте ошибки
3. Убедитесь что backend запущен:
   ```bash
   curl http://localhost:4001/health
   ```

## Перезапуск:

```bash
# Терминал 1 - Backend
cd backend
npm run dev

# Терминал 2 - Frontend  
cd frontend
npm run dev
```

## Если всё ещё пусто:

Возможно нужно пересобрать проект:
```bash
cd frontend
rm -rf .next
npm run dev
```
